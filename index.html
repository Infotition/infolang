Installation

This Project uses npm as it's package manager. Make sure you have installed it.

You can install infolang with npm:

npm install infolang

Getting Started

Now you have access to the run function.

import run from 'infolang';

const out = run(
  [
    'def n = 20',
    'for i in range(1, n) {',
    '  if ( (i mod 15) is 0 ) {',
    '    print("FizzBuzz")',
    '  }',
    '  else if ( (i mod 3) is 0 ) {',
    '    print("Fizz")',
    '  }',
    '  else if ( (i mod 5) is 0 ) {',
    '    print("Buzz")',
    '  }',
    '  else {',
    '    print(i)',
    '  }',
    '}',
  ].join('\n');

  console.log(out);
)
